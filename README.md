# ☁️ Cloud Security Lab

## 👨‍💻 About

This repository documents my **hands-on learning and practical implementation of cloud security concepts using AWS**.

The focus is on identifying **real-world misconfigurations**, understanding their impact, and applying **security best practices** to fix them.

---

## 🎯 Objective

* Build strong fundamentals in cloud security
* Gain hands-on experience with AWS services
* Understand and fix security misconfigurations
* Develop skills required for Cloud Security roles

---

## 🛠️ Topics Covered

* IAM (Identity & Access Management)
* S3 Security (Storage & Data Protection)
* Monitoring & Logging (CloudTrail, CloudWatch)
* Incident Response Basics
* Security Best Practices (Least Privilege, MFA)
